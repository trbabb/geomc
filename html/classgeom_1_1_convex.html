class geom::Convex< T, N >

Virtual class describing convex shapes in N-dimensional space.

Member Function Documentation

virtual point_t convexSupport ( point_t  d) const
pure virtual

Returns the point on the surface of this convex shape that is furthest along direction d (i.e., has the highest dot product with d).

All shapes which implement this function automatically support geometrical intersection tests with any other Convex object.

Parameters
dDirection along which to find a support plane.
Returns
A point on the surface of this convex shape.

bool intersects ( const Convex< T, N > &  other) const
inline
Returns
True if and only if this convex shape overlaps other; false otherwise.

Note that some subclasses have overloaded versions of this function for specific shapes which may offer better performance. When handling pointers to Convex objects, it may be adviseable to cast to the derived class if it is known.
