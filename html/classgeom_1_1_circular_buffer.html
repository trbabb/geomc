Detailed Description

template<typename T, index_t N>
class geom::CircularBuffer< T, N >

A lightweight circular buffer.

A circular buffer can accommodate adding elements to either the front or the back of the list in constant time.

Template Parameters
TElement type.
NStatic capacity of the buffer. Adding more than this number of elements to the buffer will incur a heap memory allocation. To always use heap allocation, pass zero to this parameter.
