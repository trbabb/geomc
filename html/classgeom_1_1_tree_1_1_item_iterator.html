Detailed Description

template<typename NodeItem, typename LeafItem>
template<bool Const>
class geom::Tree< NodeItem, LeafItem >::ItemIterator< Const >

An optionally-const iterator over the leaf items of a tree.

Dereferencing this iterator produces a LeafItem object.

Items are listed in depth-first order.

Template Parameters
ConstWhether this iterator refers to a const item or not.
