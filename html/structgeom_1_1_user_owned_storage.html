Detailed Description

template<typename T, index_t N>
struct geom::UserOwnedStorage< T, N >

Array storage with templated static or dynamic size, acting as a thin, templated wrapper around a user-owned array.

Template Parameters
TElement type.
NSize of the array, or 0 for dynamic size.

This class is mainly used for when ownership semantics are templated.
